# Personal Submission Form

## Overview
This project is a **PHP-based form submission system** designed to collect and display user information. It was created during the first year of learning web development and PHP. The form captures **personal information, educational background, eligibility details, competition performance, and general questions**. Submitted data is displayed in structured **HTML tables** after form submission.

## Features
- Collect **personal details**: name, gender, email, phone, date of birth.  
- Capture **educational background**: school, grade, combination, location, marks.  
- Record **eligibility information**: age, languages, extracurricular activities, projects, and project URLs.  
- Track **competition-related data**: week of course, typing rate, time, country.  
- Store **general information**: favorite color, description, and details of limitations.  
- Display all submitted information in a **structured table format**.  
- Built using **PHP and HTML**, with basic **CSS styling**.

## Technologies Used
- **PHP** – for handling form submission and dynamic content generation  
- **HTML** – for form structure and tables  
- **CSS** – for styling tables and page layout
